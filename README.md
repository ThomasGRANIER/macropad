# Macropad Electron

Une interface graphique pour ton macropad série, construite avec Electron.

## Fonctionnalités
- Notification système
- Support de `xdotool`, `firefox`, etc.
- Interface graphique responsive pour visualiser les macros
- Intégration dans la barre système

## Lancer l'app
```bash
npm install
npm start
```

## Modifier les macros
Édite simplement le fichier `config.json`.
